# Study Roadmap

## Goal

Build enough understanding to:

- explain YOLO clearly in an interview
- run a basic YOLOv8 project
- fine-tune on a custom dataset later

## Day 1

- Read `01_yolo_fundamentals.md`
- Understand one-stage detection, IoU, NMS, mAP

## Day 2

- Read `02_yolov8_usage.md`
- Install dependencies
- Run `scripts/check_env.py`

## Day 3

- Run `scripts/predict_image.py` on a sample image
- Understand inference outputs

## Day 4

- Read `03_interview_qa.md`
- Practice answering the questions aloud

## Day 5

- Read dataset YAML examples from Ultralytics docs
- Run `scripts/validate_model.py`

## Day 6

- Run `scripts/train_custom.py` with a small dataset
- Observe output directories and metrics

## Day 7

- Summarize:
  - what YOLO is
  - why YOLOv8 is practical
  - how training/inference work
  - how you would use YOLO in a real project
